# Service Worker 

- It is a js script that gets registered with the browser 
- it stays registered with the browser even when offline 
- can load content even with no connection 
- They canno directly access the DOM 
- It is a programeable network proxy 
- terminated when not being userd 
- makes use of promises 
- requires HTTPS unless on localhost

# Usecases
- Caching assets & API calls 
- Push notifications
- Background data sync/preload
- Used in progressive web apps 

# Lifecyle and events
Register -> Install -> Activate
