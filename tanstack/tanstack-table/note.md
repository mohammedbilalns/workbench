Headless Ui library for building tables and datagrids 

## Headless Ui ? 
Term for libraries and utilities that provide the logic , state and processing and api for ui elements and interactions, but do not provide markup, styles, or pre build implementations. 

## Core objects and Types

### Column defs 
- objects used to configure column and its data model , display templates and more 
### Table 
- Core table object containing both state and api 
### Table data 
- core data array we provide the table 
### Columns 
- irrors its respective column defs and also provides column specific apis
